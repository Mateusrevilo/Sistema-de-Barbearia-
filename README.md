# Barber Essence

Site institucional da **Barbearia Essence** — onde o estilo encontra a precisão.

## Sobre o projeto

Página web responsiva para apresentar a barbearia, com seções de apresentação, serviços, depoimentos e contato.

## Tecnologias

- **HTML5**
- **CSS3**
- **Bootstrap 5.3**
- Fontes: Inter, Montserrat, Saira, Varela (Google Fonts)

## Estrutura

```
Barbearia/
├── img/           # Imagens (logo, serviços)
├── index.html     # Página principal
└── styles.css     # Estilos customizados
```

## Seções

- **Home** — Banner e slogan
- **Quem somos** — Apresentação da barbearia
- **Serviços** — Corte simples, Corte + barba, Corte + pintura (com preços)
- **Depoimentos** — Avaliações de clientes
- **Contato** — Formulário (nome, e-mail, mensagem)

## Como usar

1. Abra o arquivo `Barbearia/index.html` no navegador.
2. Ou sirva a pasta com um servidor local (ex.: Live Server, `python -m http.server`).
